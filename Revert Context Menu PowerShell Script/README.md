# Revert Context Menu PowerShell Script

* This package reverts the "Right Click Context Menu" to the "Classic Windows" version

## Intune Platform Script Properties

1. Script Settings 
    * Run this script using the logged-on credentials: Yes
    * Enforce script signature check: No
    * Run script in 64-bit PowerShell Host: Yes
2. Assignment
    * Group Type: User
